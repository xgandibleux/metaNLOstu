# Non Linear Optimization (student version)
An example of an application of metaheuristics to the domain of Optimization. 

In this example, a simple Genetic Algorithm (GA) is designed and implemented in Julia for aiming to find the optimum of a non-linear function.

## Instructions to install and run the code
Assumption: Julia is properly installed on your computer (otherwise see [here](https://julialang.org/install/)).
- download all this repository from GitHub
- in a terminal, move into the downloaded folder, then into the `NonLinearFunction` folder
- invoke Julia
- in the Julia REPL, invoke `include("main.jl")`

Tested on macBook Pro under macOS v14.6, with Julia 1.10 using packages
- Random
- Statistics
- PyPlot
